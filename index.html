Featured Project

Strata Design System —
built from conviction.

A production-grade design system built in React, TypeScript, and Tailwind v4. Every decision from token architecture to motion accessibility reflects how I think about building things that last.

Token Architecture

Earthy, WCAG AA compliant color system with semantic tokens, paired with a dual-font typography scale that separates hierarchy from aesthetics.

Component System

Heading, Body, Button, and Animate components built with forwardRef, semantic HTML separation from visual styling, and full TypeScript coverage.

Motion & Accessibility

Framer Motion animation system with useReducedMotion support. Six animation variants with configurable triggers. Accessibility is baked in, not bolted on.
